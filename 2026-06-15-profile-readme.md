## Проекты

### 📚 [Библиотека возражений и кейсов для B2B-продаж](./README.md)

Таблица готовых ответов на типичные возражения клиентов с реальными кейсами и проверяемыми источниками + веб-интерфейс для поиска и фильтрации. Решает задачу: вместо 10-15 минут поиска подходящего примера на звонке — готовый ответ с цифрами и ссылкой за секунды.

Сделано с помощью Claude Code: ресёрч кейсов, структурирование данных, веб-интерфейс, документация — всё через диалог с агентом, без написания кода руками.
