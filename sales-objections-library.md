# Библиотека возражений (B2B-продажи обучения)

Справочник для работы с возражениями ЛПР и сотрудников компаний-заказчиков в переговорах по корпоративному обучению.

**Как пользоваться:**
- Поиск по таблице — Ctrl+F (ключевые слова из возражения)
- При появлении нового возражения — добавляйте строку в таблицу
- Колонка «Кейс/цифра» — заполняйте реальными примерами из своих сделок или из ресёрча
- Колонка «Источник» — ссылка на кейс/файл/исследование, подтверждающее «Кейс/цифра»

## Как пополнять

1. После каждой сделки — добавлять новые формулировки возражений, с которыми реально столкнулись
2. В колонку «Кейс/цифра» вписывать конкретные примеры: название компании (если можно), результат, цифры
3. В колонку «Источник» — ссылку на кейс, файл в `cases/` или исследование, подтверждающее цифру
4. При повторении одного и того же возражения в разных формулировках — объединять в одну строку, фиксируя варианты формулировок
