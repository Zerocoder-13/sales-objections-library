# Библиотека возражений и кейсов для B2B-продаж — результат за 30 дней

Презентационные материалы для внутренней демонстрации (руководство отдела продаж Zerocoder).

## До

- Таблица `sales-objections-library.md`: 12 строк-формулировок возражений, колонка «Кейс/цифра» = «заполнить» во всех строках (0/12)
- Менеджер готовился к звонкам и КП без подтверждённых кейсов и источников — аргументы были, цифры под рукой не было

## После

- 13 строк (добавлена №13 в ходе функционального теста), **12 из 13** заполнены реальными кейсами и рабочими источниками
- 4 файла кейсов в `cases/`: Squares.kz, Инвитро, ИИ-разработчики, сам проект
- `research-queue.md` — список открытых тем для дальнейшего ресёрча (по запросу)
- `templates/new-objection-row-template.md` — шаблон для новых записей с чек-листом

### Пример «до / после» (строка №2)

| | До | После |
|---|---|---|
| Возражение | «У сотрудников нет времени на обучение» | без изменений |
| Кейс/цифра | *заполнить* | Инвитро: тренинг для 25 топ-менеджеров — подготовка отчётов ускорилась в 2-3 раза, презентации на 50% быстрее |
| Источник | *заполнить* | https://ya.zerocoder.ru/kak-my-pomogli-invitro-uskorit-rabochie-processy-s-pomoshhju-ii/ |

## Замер времени (функциональный тест, день 22-24)

Новое возражение с реального звонка → готовая строка с ответом, кейсом и источником в таблице:

- **Время агента:** ~1 минута (поиск похожего + формулировка + обновление таблицы)
- **Ручной поиск кейса и формулировка вручную:** ориентировочно 10-15 минут

## Инструменты и затраты

- Claude Code Pro (уже оплаченная подписка) + бесплатные инструменты
- Без новых платных подписок и сервисов
- Ollama протестирован для локального ресёрча — не используется для фактов в библиотеке (риск выдуманных данных без доступа в интернет)

## Метрики проекта

- **Количественно:** 0/12 → 12/13 строк с кейсом и источником
- **Качественно:** каждая цифра проверяема по ссылке, без выдуманных данных
- **Функционально:** прогон на реальном возражении — пройден (см. `reports/2026-06-15-scenario-functional-test-report.md`)
- **Субъективно:** проверяется через месяц использования библиотеки на звонках
