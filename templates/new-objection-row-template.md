# Шаблон: новая строка для библиотеки возражений

Используется для добавления нового возражения в `sales-objections-library.md`.

## Настройка в Obsidian (один раз)
1. Settings → Core plugins → включить «Templates»
2. Settings → Templates → Template folder location → указать `templates`
3. При добавлении новой строки: Ctrl+P → «Insert template» → выбрать этот файл

## Шаблон строки

```
| [№] | «[текст возражения]» | [контекст / когда возникает] | [ответ / аргумент] | [кейс/цифра или *заполнить*] | [источник или *заполнить*] |
```

## Чек-лист перед добавлением

- [ ] Формулировка возражения близка к тому, как говорит клиент
- [ ] Указан контекст — на каком этапе переговоров возникает
- [ ] Аргумент предлагает конкретное действие/ответ, а не общую фразу
- [ ] Если есть кейс — указан источник: ссылка, файл в `cases/` или «личный опыт менеджера»
- [ ] Если кейса нет — добавить тему в `research-queue.md` со статусом «открыто»
- [ ] При повторении формулировки уже существующего возражения — не создавать новую строку, дополнить существующую
